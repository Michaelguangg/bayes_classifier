腾讯科技讯 11月21日，据CNN报道，7月份，恐怖组织IS相关的推特账号发出警告，暗示将对突尼斯海岸旅游胜地发动恐怖袭击。而仅仅1个月前，类似袭击就曾导致38人遇难。如果布什民间反恐黑客组织Ghost Security Group发出警告，这样的悲剧可能再次上演。Ghost Security Group组织主要由反间官员和计算机专家组成，他们监控这个推特账号已经一个多月时间。这个账号只有少数追随者，但其中许多人都是IS组织的资深成员。Ghost Security Group立即寻找情报承包商，将情报传递给相关各国政府。他们通过推特找到反恐专家麦克尔&#183;史密斯（Michael S. Smith II）。史密斯当时在国防咨询机构Kronos Advisory效力，担任国会议员的反恐顾问。他对Ghost Security Group组织提供的情报立即做出回应，并成为该组织与政府接触的渠道。据史密斯称，调查人员利用Ghost Security Group提供的情报确认了袭击地点，包括哪些人正成为袭击目标，恐怖分子如何发动袭击等。最终，数十名恐怖嫌犯被捕。史密斯说：“毫无疑问，这个组织挽救了很多人命，至少数十人。许多美国、欧洲以及中东国家的安全机构情报人员却没有提前发现这场阴谋。”FBI局长詹姆斯&#183;科米（James Comey）多次表示，IS正空前积极地利用社交媒体与西方人交战不休。史密斯说，IS成员在社交媒体上占据很大优势，这意味着他们需要Ghost Security Group这样的外部力量支持。但是为了让这些团体发挥高效作用，他们需要与管理者积极协调，以寻找和确认敌人。Ghost Security Group与松散的黑客组织“匿名者”完全不同，后者已经对IS组织宣战，宣称将瘫痪IS组织推特账号。Ghost Security Group的少数成员曾加入“匿名者”，包括其绰号DigitaShadow的领导者。DigitaShadow称，Ghost Security Group规模很小，但却更专注。他说：“我们有组织和领导者，也有许多反恐经验。我们有翻译、语言学家以及研究分析师，可以处理和分析收到的所有数据。”DigitaShadow相当于执行董事的角色，帮助组织日常活动，并向14名成员分配任务。他还负责为组织提供电子设备。Ghost Security Group也与其他组织合作，比如帮助监控恐怖分子在社交媒体上活动的CtrlSec。Ghost Security Group是法国巴黎《查理周刊》杂志社遭到恐怖袭击后组建的。DigitaShadow说，所有成员都是全职职员，分散在世界各地。即使他们都是志愿者，每天工作也长达16小时。他说：“我们首次意识到，你们（恐怖分子）可以在巴黎街头发动袭击，也可以在美国发动袭击，每个人都可能成为受害者，我们希望竭尽全力阻止他们。”DigitaShadow说，自从Ghost Security Group组建以来，他们已经瘫痪了149个IS组织宣传网站、11万个社交媒体账号以及删除6000多段宣传视频。最近巴黎遭袭后，Ghost Security Group成员试图收集攻击者的数字脚印，并确认参与袭击者的社交媒体账号。Ghost Security Group宣称已经开发出自动软件，可以识别IS的社交媒体账号。DigitaShadow说，他们已经渗透到IS的私人通信渠道，接管IS社交媒体账号，提取IP信息帮助识别和确定IS成员位置。Ghost Security Group主要致力于打击IS，但他们也针对其他伊斯兰极端组织。史密斯表示，Ghost Security Group也在沙特确认和追踪两兄弟，他们曾拍摄杀害某人的视频以示对IS的支持。Ghost Security Group能够控制那些上传执行死刑视频的推特账号，并寻找有关移动设备的信息，帮助政府确认杀手位置。今年夏季与史密斯取得联系并向官方传递消息后，Ghost Security Group改变了某些策略，行动更加符合法律规定。现在，这个组织开始收集有价值数据，然后发送给政府。DigitaShadow说：“我们的黑客袭击合法吗？绝对合法。打击IS以阻止他们的威胁和宣传属于合法行为吗？这是个巨大的灰色地带。” Ghost Security Group团队正加班加点工作，他们没有补助，但可以收到比特币捐款。尽管入不敷出，但DigitaShadow表示，他们绝不会停止。他说：“如果我们现在停下来，生命将会陷入危险。这不是一个选择，而是我们现在的生活方式。” （风帆）
