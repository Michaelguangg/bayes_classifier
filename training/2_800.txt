腾讯科技讯 据英国路透社报道，Anonymous并非首个向极端组织“伊斯兰国”发起攻击的黑客组织。“伊斯兰国”的同情者通过社交网络进行宣传并开展人员招募，此举招致激进人士越来越多的回击，他们致使部分网站瘫痪并对其他网站进行渗透。松散的黑客组织Anonymous是最近一个加入这一反击阵营的激进派人士，其成员称本周已经使得数千个亲“伊斯兰国”的Twitter账号不能使用。不过，据称其他黑客组织从事打击“伊斯兰国”网络攻势的时间更长。一个向美国政府反馈信息的组织就表示，从今年1月份开始，其已经攻击了数万个亲“伊斯兰国”的Twitter账号，其成员还伪装成应聘者从支持“伊斯兰国”运营的所谓“Dark Web”网络上获得信息。Ghost Security Group匿名负责人表示：“我们更多是扮演搜集情报的工作。” Ghost Security Group是一个志愿者组织，其已经通过美国国会反恐顾问迈克尔&#8226;史密斯(Michael S. Smith II)将数据转交给美国联邦调查局(FBI)等其他机构。史密斯表示，该组织的渗透活动已经向美国政府提供一些可以采取行动的信息，配合相关信息向Twitter进行投诉，有助于把“伊斯兰国”支持者赶到其他地方。史密斯说：“美国相关机构对于外来的支持表示感谢，我因此不断收到反馈。”美国退役将军戴维&#8226;彼得雷乌斯(David Petraeus)近期对《外交政策》(Foreign Policy)杂志表示，他已经看过了史密斯的数据，并认为其对于正在参与反恐活动的人士而言具有相当高的价值。美国联邦调查局拒绝就此置评。史密斯同时还担任美国国家安全部门承包商Kronos Advisory的首席运营官(COO)。史密斯表示，Ghost Security Group今年6月份联系到他，并提供了针对突尼斯将发起袭击的内部聊天截图。史密斯向有关部门传递了这一信息，并帮助他们立即捣毁了一处武装分子巢穴。上周五发生在法国巴黎的系列恐怖袭击事件使得打击圣战分子的网络活动进一步升级，但是通过网络对其进行对抗的活动仍面临不少的困难。普通民众进行黑客行为和DoS攻击，其目的是使计算机或网络无法提供正常的服务，但无论其针对的目标是什么，这种行为都是非法的。包括冒充应聘极端分子在内的复杂渗透活动，看起来是好心帮忙，其实可能加大美国政府以及其同盟国家进行反恐的难度。美国退役空军上将麦克&#8226;海登(Mike Hayden)曾担任过美国国家安全局(National Security Agency)以及美国中央情报局(Central Intelligence Agency)的负责人。被问及这些情报机构对于Ghost Security Group的行为是否会表示感谢时，海登表示：“从官方层面上，绝对不可能。不过，美国的法律和政府非常具有强迫性。我认为那些政府的人士正在偷着乐，就和我一样。”志愿者能做的最简单的事情就是向Twitter、谷歌(微博)的YouTube以及Facebook进行申诉，称他们服务上的有些账号支持恐怖主义。激进人士表示这三家公司在过去一年已经进行更多响应，但均拒绝透露这方面的具体情况。Ghost Security Group称，Facebook今年禁止在其网站上对“恐怖主义”进行任何赞美活动。YouTube目前则可以在数小时之内将暴力视频撤下。在Twitter上，Ghost Security Group和旗下一个组织传播存在问题的账号名单。看到这些名单的普通用户可以向Twitter进行举报，致使这些账号关闭，这要比其他组织单方面投诉快得多。智库布鲁金斯学院研究“伊斯兰国”的专家J&#8226;M&#8226;伯格(J.M. Berger)表示：“越来越多的账号被查封。我认为大部分这样的报告是通过Anonymous和 Ghost Security来完成的。不过，也有其他方面的功劳，例如Counter Extremism Project和Sawab Center，他们也在做这方面的工作。”伯格表示，这样的努力有助于将“伊斯兰国”Twitter账号的受众保持在一个较窄的范围，这一行为具有积极意义。伯格和其他安全专家表示，在Twitter上承受压力是“伊斯兰国”将其宣传精力转移至私密安全即时通信应用Telegram的原因之一，其在该平台上开通服务渠道，使得一位参与者可以接触到数千受众。绰号为“DigitaShadow”的The Ghost Security Group领导人表示，该组织仍然在Telegram上搜集信息。DigitaShadow还表示，大部分网站是在该组织研究员通知网站托管服务商后停止工作的。该组织只有在少数时候采用了DoS攻击模式。不过，DigitaShadow表示，通过史密斯传达的指导意见通常要求不使用这样的暴力破解(Brute Force)方法。DigitaShadow说：“我们已经放弃了DoS攻击，更多的则是转向情报收集。”（李路）
